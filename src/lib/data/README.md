## Country List

The country registry is generated from
[ISO 3166 Countries with Regional Codes](https://github.com/lukes/ISO-3166-Countries-with-Regional-Codes).
AirTrail uses ISO 3166-1 alpha-2 codes as its country identifiers.

Kosovo is added using the EU corporate country vocabulary's user-assigned `XK`
code. Its map geometry comes from the Eurostat GISCO NUTS 2024 level-0 data set,
which also supplies the Serbia geometry that excludes Kosovo.

Palestine uses its ISO `PS` code. Taiwan uses its ISO `TW` code. Natural Earth
Admin 0 supplies coordinated China and Taiwan geometries because the GISCO
world-country data does not provide a separate Taiwan feature. These entries are
travel territories; their inclusion does not express a position on sovereignty
or legal status.
